ECMAScript 6: 在下一个javascript版本中有哪些新东西

2016-10-30

简介

你可能已经听到关于ECMAScript6 (或者ES6), 作为下一个JavaScript的版本, 它也有一些很棒的新特性。这些特性有不同程度的复杂性，并在脚本和应用程序非常有用。在这篇文章里，我们将会讨论哪些经过精心挑选并且在你每天javascript coding 中使用的功能。
请注意，现在浏览器会很好的支持这些新的ES6功能，也许支持的情况会有所不同。如果你需要支持缺少许多ES6功能的浏览器，我会寻找解决方案，今天可以帮助你开始使用ES6。
大部分的代码都带有一个外部的“运行代码”链接，这样就可以看到代码并使用它。

变量

let
你已经使用过var 声明变量。现在可以更好的使用let。细小之处在于作用域。var结果取决于函数作用域，let的作用域范围则只是它所在的代码块。

if(true) {
    let x = 1;
}
console.log(x); // undefined

这可以使你的代码更加干净，从而减少更多的变量。就拿这个经典的数组循环来说：

for(let i = 0, l = list.length; i < l; i++) {
   // do something with list[i]
}

console.log(i); // undefined

通常，我们这么使用的时候，例如，在循环中定义变量i和在循环外定义结果都是一样。但用let，你可以放心地再次声明i，因为它只能在自己的块级作用域内定义和使用。

const
还有另一种方式来声明块级作用域的变量。这就是const，你只需要声明一个只读参考值。必须直接赋值一个变量。如果你试图改变变量，或者如果你没有直接给变量赋值，那么你会得到一个错误：

const MY_CONSTANT = 1;
MY_CONSTANT = 2 // Error
const SOME_CONST; // Error

请注意，您仍然可以更改对象属性或数组内容：

const MY_OBJECT = {some: 1};
MY_OBJECT.some = 'body'; // Cool

箭头函数

箭头函数是一个添加到JavaScript中非常棒的功能。他们缩减了代码。在这篇文章中我们很早就引进箭头函数，这样我们就可以在后面的例子中使用它们。下面的代码片段显示一个箭头函数，和我们所熟悉的ES5相同功能的代码：

let books = [{title: 'X', price: 10}, {title: 'Y', price: 15}];

let titles = books.map( item => item.title  );

// ES5 equivalent:
var titles = books.map(function(item) {
                return item.title;
             });

运行代码

如果我们看一下箭头函数的语法，这里并没有function关键字，剩下的0或者其他的参数、”箭头”(=>)和函数表达式，return则被隐性添加进去。
如果有0个或者多个参数，必须添加括号:

// No arguments
books.map( () => 1  ); // [1, 1]

// Multiple arguments
[1,2].map( (n, index) => n * index  ); // [0, 2]

如果你需要更多的逻辑和空格，把函数表达式放到{...}块里：

let result = [1, 2, 3, 4, 5].map( n => {
        n = n % 3;
        return n;
    });

运行代码

不仅箭头功能意味着更少的字符输入，他们也和常规的函数表现不同。箭头函数表达式在执行上下文中指定了this和arguments。这意味着你可以摆脱像var that = this这样难看的语句，你将不需要将函数绑定到正确的上下文中。下面是一个例子（注意this.title与that.title在ES5版）：

let book = {
    title: 'X',
    sellers: ['A', 'B'],
    printSellers() {
        this.sellers.forEach(seller => console.log(seller + ' sells ' + this.title));
    }
}

// ES5 equivalent:
var book = {
    title: 'X',
    sellers: ['A', 'B'],
    printSellers: function() {
        var that = this;
        this.sellers.forEach(function(seller) {
            console.log(seller + ' sells ' + that.title)
        })
    }
}

运行代码

String

方法

一些便利的方法已经被添加到String原型中,它们中的大部分已经删除了通过用indexOf()方法实现了相同的功能:

'my string'.startsWith('my'); //true
'my string'.endsWith('my'); // false
'my string'.includes('str'); // true

简单而有用的，另一个方便的方法已经被添加到作为创建一个重复的字符串中:

'my '.repeat(3); // 'my my my '

模板文字提供一个清洁的方式来创建字符串，并执行字符串插值。您可能已经熟悉了这种 语法;它是基于美元符号和花括号$ {..}。模板文字是由反引号括起来。这里有一个快速演示：

let name = 'John',
    apples = 5,
    pears = 7,
    bananas = function() { return 3;  }

    console.log(`This is ${name}.`);

    console.log(`He carries ${apples} apples, ${pears} pears, and ${bananas()} bananas.`);

    // ES5 equivalent:
    console.log('He carries ' + apples + ' apples, ' + pears + ' pears, and ' + bananas() +' bananas.');

运行代码

在上面的形式，相对于ES5，它们仅仅是为了方便字符串连接。然而，模板文本可以也可用于多行的字符串。请记住，空格也是字符串的一部分：

let x = `1...
2...
3 lines long!`; // Yay

// ES5 equivalents:
var x = "1...\n" + 
"2...\n" +
"3 lines long!";

var x = "1...\n2...\n3 lines long!";

运行代码

Array

数组对象现在有一些新的静态类的方法，以及数组原型的新方法。
首先，Array.from创建一个从类数组和继承对象的Array实例。类数组的例子包括：

  • arguments 在函数中；
  • nodeList通过document.getElementsByTagName()返回；
  • 新的Map 和Set 数据结构；
let itemElements = document.querySelectorAll('.items');
let items = Array.from(itemElements);
items.forEach(function(element) {
    console.log(element.nodeType)
});

// A workaround often used in ES5:
let items = Array.prototype.slice.call(itemElements);

运行代码

在上面的例子中你能看到items数组有forEach方法，这些并不是itemElements集合中可以用的.
一个有趣的属性在Array.form中是第二个参数选项mapFunction,它允许你创建新的映射数组在单个调用方法中

let navElements = document.querySelectorAll('nav li');
let navTitles = Array.from(navElements, el => el.textContent)

运行代码

然后，我们有Array.of,表现的更像是Array构造器,它修正了当数字作为参数的特殊情况,
Array.of里的返回值更像new Array()，然后，在大部分情况，你应该使用数组字面量。

let x = new Array(3); // [undefined, undefined, undefined]
let y = Array.of(8); // [8]
let z = [1, 2, 3]; // Array literal

最后一些比较重要的是，一些方法已经加入到Array原型中，我认为这里面find将会是最受开发者欢迎的方法.

  • find返回符合回调函数里为true的第一个元素
  • findIndex返回符合回调函数里为true的第一个索引值

  • fill “覆盖”数组的给定参数的元素

    [5, 1, 10, 8].find(n => n === 10) // 10

    [5, 1, 10, 8].findIndex(n => n === 10) // 2

    [0, 0, 0].fill(7) // [7, 7, 7]
    [0, 0, 0, 0, 0].fill(7, 1, 3) // [0, 7, 7, 7, 0]

Math

一些新的方法已经加入到Math对象中.

  • Math.sign返回数字的标志比如1,-1或0.
  • Math.trunc返回整数值.

  • Math.cbrt返回立方根的值.

    Math.sign(5); // 1
    Math.sign(-9); // -1

    Math.trunc(5.9); // 5
    Math.trunc(5.123); // 5

    Math.cbrt(64); // 4

如果你想学到更多关于数字的运算方法请点击

操作符

操作符...是一种非常方便的语法在特定的地方扩展数组中的元素，就像参数在函数中的调用，展示一些例子可能是最好的方式来证明他们是多么的有用.
首先，让我们看看怎么在另一个数组中扩充数组中的元素：

let values = [1, 2, 4];
let some = [...values, 8]; // [1, 2, 4, 8]
let more = [...values, 8, ...values]; // [1, 2, 4, 8, 1, 2, 4]

// ES5 equivalent:
let values = [1, 2, 4];
// Iterate, push, sweat, repeat...
// Iterate, push, sweat, repeat...

运行代码

当用参数调用函数的时候操作符也很有用

let values = [1, 2, 4];

doSomething(...values);

function doSomething(x, y, z) {
       // x = 1, y = 2, z = 4

}

// ES5 equivalent:
doSomething.apply(null, values);

运行代码

就像你看到的一样,这节省了我们的常用fn.apply()方法。语法非常灵活,因为传播算子可以在参数列表的任何地方使用。这意味着下面的调用会产生同样的结果:

let values = [2, 4];
doSomething(1, ...values);

运行代码

我们可以把运算符应用到数组和参数上。事实上，我们可以把他应用到所有的对象上，就像这样一个NodeList:

let form = document.querySelector('#my-form'),
    inputs = form.querySelectorAll('input'),
    selects = form.querySelectorAll('select');

let allTheThings = [form, ...inputs, ...selects];

运行代码

现在，allTheThings是一个二维数组包括<form>在内的节点它里面的<input>和<select>子节点

解构

解构提供了一种方便的方法从对象或数组来提取数据。首先,可以使用一个数组来展示：

let [x, y] = [1, 2]; // x = 1, y = 2

// ES5 equivalent:
var arr = [1, 2];
var x = arr[0];
var y = arr[1];

使用这个语法,可以将多个变量赋值。不错的作用是,您可以很容易地交换变量值:

let x = 1,
    y = 2;

[x, y] = [y, x]; // x = 2, y = 1

运行代码

解构也可以用在对象上，确定有可以匹配的属性：

let obj = {x: 1, y: 2};
let {x, y} = obj; // x = 1, y = 2

运行代码

你也可以使用这种机制改变变量的名字：

let obj = {x: 1, y: 2};
let {x: a, y: b} = obj; // a = 1, b = 2

另外有趣的地方是可以模拟多个返回值：

function doSomething() {
   return [1, 2]
}

let [x, y] = doSomething(); // x = 1, y = 2

运行代码

解构可用于函数参数的默认值赋值。使用对象字面量,你可以模拟参数。

function doSomething({y = 1, z = 0}) {
   console.log(y, z);
}
doSomething({y: 2});

参数

默认值
在ES6,定义函数参数的默认值是可以的。语法如下:

function doSomething(x, y = 2) {
    return x * y;

}

doSomething(5); // 10
doSomething(5, undefined); // 10
doSomething(5, 3); // 15

运行代码

看起来很干净,对吧?我确信在ES5之前你需要填充一些参数:

function doSomething(x, y) {
   y = y === undefined ? 2 : y;
      return x * y;

}

缺省默认参数值则会出现underfind
省略参数
我们已经学习过运算符，省略参数是非常灵活的，它还使用...语法和允许您将末尾参数存储在一个数组:

function doSomething(x, ...remaining) {
   return x * remaining.length;
}

doSomething(5, 0, 0, 0); // 15

运行代码

模块

模块肯定是一个受欢迎的加入到JavaScript中。我认为这是值得挖掘ES6特点。
任何严格的javascript项目都使用了模块化的方法－也许就像“模块模式”或者更多广泛使用的AMD和Common Js.然而，浏览器不是任何特性,你总是需要一个构建步骤或AMD或CommonJS模块加载器,使用工具来处理模块包括RequireJS，Browserify和Webpack.
ES6规范包括一个新的语法和加载程序模块的机制,如果在未来你想使用模块编写,这是你应该使用的语法,现代构建工具支持这种格式,可能通过一个插件,所以你应该很好使用了(别担心，我们在后面会讨论如何编译ES6).
现在，在ES6的模块语法上，模块设计通过export和import两个关键字引入.现在，用这两个例子让我们看看如何通过这两个方法引入模块.

// lib/math.js

export function sum(x, y) {
    return x + y;
}
export var pi = 3.141593;


// app.js

import { sum, pi  } from "lib/math";
console.log('2π = ' + sum(pi, pi));

就像你看到的那样，这可以有多个export表述，每个必须显式状态导出的值的类型(在这个例子中，是function 和 var)

import声明在这个例子中使用的语法(类似重构)明确定义引入的模块,引入模块作为一个整体,可以使用*通配符,给模块链接作为本地名称的关键字.

// app.js

import * as math from "lib/math";
console.log('2π = ' + math.sum(math.pi, math.pi));

模块系统功能default的出口,这可以是一个函数,在模块中引入一个默认的值，你只需要一个本地名称(重构前)

// lib/my-fn.js

export default function() {
       console.log('echo echo');

}

// app.js

import doSomething from 'lib/my-fn';
doSomething();

请注意,import语句是同步的,但是该模块代码在所有的依赖代码执行前不会执行。

Classes

Classes 是ES6非常棒的特性，一些人认为他们违背JavaScript的原型的本质,而其他人认为他们降低门槛对初学者和人来自其他语言,他们帮助人们编写大型应用程序。在任何情况下,他们是ES6的一部分。这有一个快速的介绍。
类是建立在class和constructor关键字。这是一个简短的例子:

class Vehicle {
    constructor(name) {
        this.name = name;
        this.kind = 'vehicle';
    }
    getName() {
        return this.name;
    }   
}

// Create an instance
let myVehicle = new Vehicle('rocky');

运行代码

注意,类定义并不是一个普通对象;因此,类成员之间没有逗号

通过类来创建一个实例，你必须通过new关键字,从class来继承一个基类，使用extends:

class Car extends Vehicle {
    constructor(name) {
        super(name);
        this.kind = 'car'
    }
}

let myCar = new Car('bumpy');

myCar.getName(); // 'bumpy'
myCar instanceof Car; // true
myCar instanceof Vehicle; //true
